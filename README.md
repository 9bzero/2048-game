# 2048

Slide the tiles, merge the numbers, reach **2048**.

## How to play

Use arrow keys (or swipe on mobile) to slide all tiles. When two tiles with the same number collide they merge. Reach 2048 to win — or keep going for a higher score.

## Features

- Keyboard and touch swipe support
- Smooth tile slide and merge animations
- Best score saved to localStorage
- Undo last move
- New game button
